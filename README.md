# cppBlogGen
a simple, naive blog content generator written entirely on my 2008 MacBook Pro.

## Directory structure
```
src
	- includes
		- HTMLElements.hpp
		- MarkdownParser.hpp
		- CommandParser.hpp
		- ConfigParser.hpp
	- build
		- BlogGen.cpp
demo
	- demo-site
		- layouts
		- themes
		- contents
			- posts
				- sample-post.md
			- pages
				- sample-page.md
		- config.cbgconf
```
